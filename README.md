A tsundere Japanese maid AI assistant with voice interaction and emotion display.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Ensure Ollama is running with llama3.2
ollama pull llama3.2

# Run Sakura
python -m sakura
```

## Features

- Local AI powered by Ollama (llama3.2)
- Voice input with faster-whisper
- Text-to-speech with Edge TTS (Nanami voice)
- Emotion-based character images
- Conversation memory across sessions
